Le sept septembre deux mille quinze, Cher monsieur, Nous allons passer a la derniere phase suivante de notre stratageme. A partir de maintenant, nous utiliserons une des cachettes secretes de la societe pour communiquer. Il s'agit d'un ancien garde meuble. Il se trouve au cinq boulevard Lewis Carroll. Vous y trouverez tout un tas de documents je vous laisse faire le tri. Pour y acceder il vous faut le numero du box et le code du cadenas. Comme ce mode de chiffrement n'est pas fait pour les nombres, je les ai ecrits ci-dessous, en les chiffrant d'une autre maniere. Ce code ne doit en aucun cas etre ecrit ou divulgue a un tiers. Vous devez les calculer de la maniere suivante : multipliez les lettres de votre prenom avec A fait un, B fait deux, etc. Faites le produit du nombre obtenu et de quarante-sept et otez dix-neuf. Le resultat obtenu sera votre cle. Afin de localiser le box, prendre le grand nombre ci dessous et y retrancher cinquante. Le resultat obtenu doit etre divise par votre cle. Cela donne cinq chiffres, les deux premiers forment l'identifiant du box, les suivants le code du cadenas.
